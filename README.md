# Sales Data Analysis using SQL

## Project Overview
This project analyzes a sales dataset using SQL to extract meaningful insights.

## Key Insights
- Calculated total revenue using aggregation
- Identified top customers based on sales
- Analyzed product line performance
- Counted total number of orders

## Tools Used
- SQL (SQLite)
- CSV dataset

## Files Included
- queries.sql → SQL queries used
- sales_data.csv → dataset
- screenshots → output results

## Conclusion
This project demonstrates strong SQL skills including filtering, aggregation, grouping, and business insight generation.
