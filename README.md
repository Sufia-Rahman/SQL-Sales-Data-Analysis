# 📊 SQL Sales Data Analysis

## 🔍 Project Overview
This project analyzes global sales data using SQL to extract meaningful insights such as revenue trends, customer behavior, and product performance.

---

## 🎯 Objectives
- Analyze total sales and revenue
- Identify top-performing products
- Segment customers based on revenue
- Find high-value customers using advanced SQL

---

## 📁 Dataset
- Sales dataset containing:
  - Order Number
  - Sales
  - Customer Name
  - Product Line
  - Country
- Data cleaned and analyzed using SQL

---

## 🛠 Tools Used
- SQL (SQLite)
- Excel (for dataset)
- GitHub (for project hosting)

---

## 📌 Basic Queries
- View data
- Total revenue calculation
- Total number of orders
- Sales by product line
- Top customers

---

## 🚀 Advanced Analysis

### 1. Customer Segmentation (CASE)
- Classified customers into:
  - High Value
  - Medium Value
  - Low Value

### 2. Subquery Analysis
- Identified customers with above-average revenue

### 3. Product Performance
- Top-performing product lines based on revenue
